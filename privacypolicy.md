# Privacy Policy

**Last Updated:** April 27, 2026

This Privacy Policy explains how our bot/service collects, uses, and protects your information.

---

## 1. Information We Collect

When you use our bot/service, we may collect the following:

### Discord Data
- Server (Guild) IDs  
- Channel IDs  
- User IDs  
- Message content (only when required for features like moderation, logging, or ticket systems)

### Configuration Data
- Server settings  
- Automod settings (anti-link, anti-invite, blacklist words, etc.)  
- Ticket system data  
- Feature preferences per server  

### Logs
- Command usage logs  
- Error logs  
- Moderation action logs (if enabled)

---

## 2. How We Use Data

We use collected data to:
- Provide bot functionality (moderation, tickets, automod, etc.)
- Improve performance and stability
- Prevent abuse and ensure security
- Fix bugs and errors

We **do not sell, trade, or share** your data with third parties.

---

## 3. Data Storage

- Data is stored using databases like **SQLite** or secure storage systems  
- Data is kept only as long as necessary for bot functionality  
- Server owners may reset data using available commands (e.g. `/ticket reset`)

---

## 4. Data Sharing

We only share data if:
- Required by law  
- Necessary to protect the bot/service from abuse or attacks  

---

## 5. User Control

Server administrators may:
- Reset server data  
- Disable certain features  
- Remove or reconfigure bot settings  

---

## 6. Security

We take reasonable steps to protect your data.  
However, no system is completely secure, and we cannot guarantee absolute protection.

---

## 7. Third-Party Services

Our bot may interact with third-party services such as:
- Discord API

Their privacy policies apply when using those services.

## 8. Changes to This Policy

We may update this Privacy Policy at any time.  
Continued use of the bot means you accept the updated policy.
